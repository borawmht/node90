# Node90 Firmware

MCU: PIC32MX795F512H
PHY: LAN8720A

# Project

Setup using Code Configurator v5.5.1 and Harmony 3 1.5.5. 

Build with XC32 4.60.

## Modules
- csp 3.22.6
- core 3.15.3
- zlib 1.3.1
- FreeRTOS-Kernel 11.2.0
- wolfssl 5.4.0 (build errors if using 5.7.0)
- net 3.14.2
- crypto 3.8.2
